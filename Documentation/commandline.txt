***Running GO-Elite by Command-Line***In addition to the graphical user interface, GO-Elite can be run using the command line in a terminal or DOS prompt window. This is accomplished by downloading the source code version of GO-Elite, opening a terminal or DOS prompt window, changing directories to the GO-Elite application folder and inputting the command line options. Example command-line options are given below. ***Download the GO-Elite Database (required) ***$python GO_Elite.py --update Official --species Mm --version ÊEnsMart62Plus --additional allNote: the last argument --additional (optional) retrieves all other optional gene-sets, ontologies and pathway resources available, rather than just GO and WikiPathways. ***Run GO-Elite on Input and Denominator List Directories***--Run using defaults-- $python GO_Elite.py --species Mm --input "C:/Mm_sample/input_list_small" --denom "C:/Mm_sample/denominator" --output "C:/Mm_sample"--Run explicitly--$python GO_Elite.py --species Mm --mod Ensembl --permutations 2000 --method "z-score" --zscore 1.96 --pval 0.05 --num 3 --input "C:/Mm_sample/input_list_small" --denom "C:/Mm_sample/denominator" --output "C:/Mm_sample"***Visualizing WikiPathways with User Input ID Lists***$python GO_Elite.py --input "C:/Mm_sample/input_list_small/myometrium_term_cluster_probesets.txt" --image WikiPathways --mod Ensembl --species Mm --wpid WP572Note: The user input ID list must have at least 3 columns: (1) IDs, (2) System Code, (3) positive and negative log2 fold changes. ***Update or Add Ontology Structure Support***$python GO_Elite.py --update Ontology --OBOurl http://www.geneontology.org/ontology/gene_ontology.obo Note: Replace the URL with any Ontology of interest (requires the addition of a ID-to-ontology association file in Databases/EnsMartXX/Species/gene-go. ***Results Interpretation***http://code.google.com/p/go-elite/wiki/Tutorial_GUI_versionAdditional Detailshttp://genmapp.org/go_elite/help.htm#commandline
